# ParalogICB
Systematically identification of ICB paralogs.

##  Prerequisites
Make sure the following programs/packages installed and added to your `PATH`:
- Python 3.9+
  - pandas
  - numpy
  - scipy
  - urllib
  - pathlib
  - gzip
  - glob
  - statsmodels
  - bs4
  - functools
  - cmapPy
  - rpy2
  - multiprocessing
- R 4.2.0+
  - base
  - stats
  - survival
  - biomaRt
  - TCGAbiolinks
- Blast-2.13.0+
- PAML-4.10.6
  - codeml
- pal2nal-v14
- ClustalW2-2.1
- TM-score
