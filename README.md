# ParalogICB
Systematically identification of ICB paralogs.

##  Prerequisites
Make sure the following programs/packages installed and added to your `PATH`:
- Python 3.9+
  - pandas
  - numpy
  - scipy
  - urllib
  - pathlib
  - gzip
  - glob
  - statsmodels
  - bs4
  - functools
  - rpy2
- R 4.2.0+
  - base
  - stats
  - survival
  - TCGAbiolinks
- Blast
- PAML
